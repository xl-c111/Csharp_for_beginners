# C# Naming Rules

| Thing          | Naming Style     | Example            |
| -------------- | ---------------- | ------------------ |
| Class          | PascalCase       | `UserAccount`      |
| Method         | PascalCase       | `ChangePassword()` |
| Property       | PascalCase       | `UserName`         |
| Private field  | `_camelCase`     | `_password`        |
| Local variable | camelCase        | `inputPassword`    |
| Parameter      | camelCase        | `oldPassword`      |
| Interface      | `I` + PascalCase | `IUserService`     |
| Constant       | PascalCase       | `MaxLoginAttempts` |
| Enum           | PascalCase       | `OrderStatus`      |
| Enum value     | PascalCase       | `Pending`          |
| Namespace      | PascalCase       | `MyApp.Models`     |
